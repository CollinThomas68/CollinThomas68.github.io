Les classes pour changer la largeur d'un élément

On utilise la classe "w-" associée à un pourcentage, il faut obligatoirement que cela soit 25 50 75 ou 100%
Ce qui donnera class="w-50". Il existe aussi la class auto pour garder la taille d'origine de l'élément.

On peut faire la même chose pour la hauteur avec la class "h-"
